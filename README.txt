this is OpenGL 3D racing game.
this project uses Textures, Different types of lighting and Material properties to render the scene.

you can find here the code and the jar to run the game.
in order to run the jar (from the implemented forder), place the jar file and the texture folder into natives forder in to your os folder name 
double click or press enter and you are there.


You can press on:
1. ‘UP’ - acceleration.
2. ‘DOWN’ - breaks.
3. ‘LEFT’ - rotates the steering to the left direction.
4. ‘RIGHT’ - rotates the steering to the right direction.
5. ‘l’ (L) - switches the light mode from Day mode to night mode.
6. ‘v’ (V) - switches the view point
